How this site works

Primary sources only

FBI, CDC, Census Bureau, peer-reviewed journals. No aggregators, no op-eds — only the underlying data, cited directly on every page.

Steelmanning both sides

Every opposing argument is presented with its strongest supporting evidence before being addressed. No strawmanning, no selective quotation.

Sensitivity analysis

Every assumption is tested. Results shown under multiple definitions, time windows, and denominators — so you can see how robust each conclusion actually is.

Honest about limits

Wide confidence intervals are shown, not hidden. Data gaps are named. What the evidence can't establish matters as much as what it can.
